# Black Barbershop

A modern barbershop booking application built with React and TypeScript.

## Project info

Premium barbershop booking experience for finding and booking appointments with local barbers.

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

Follow these steps:

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd blackbarbershop

# Step 3: Install dependencies
npm i

# Step 4: Start the development server
npm run dev
```

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Features

- Browse nearby barbershops
- Book appointments with preferred barbers
- Manage appointments
- Earn rewards points
- Mobile-responsive design
